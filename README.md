# Španělština od nuly

Statická webová aplikace pro úplné začátečníky ve španělštině.

## Obsah

- 10 logických lekcí od pozdravů po základní otázky.
- 300 základních slovíček s českým překladem a orientační výslovností.
- 100 základních frází s českým překladem a orientační výslovností.
- Hry: rychlý kvíz, padající slova, pexeso překladů a doplňování překladu.
- Slovník s vyhledáváním a tiskem do PDF přes systémový tisk.

## Spuštění

Otevři `index.html` v prohlížeči.

## GitHub Pages

1. Nahraj soubory `index.html`, `styles.css`, `app.js` a `README.md` do repozitáře.
2. V GitHubu otevři `Settings > Pages`.
3. Vyber větev `main` a složku `/root`.
4. Po uložení GitHub zobrazí veřejnou adresu aplikace.

## Tisk slovníku do PDF

V aplikaci otevři záložku `Slovník` a klikni na `Tisk / PDF`. V dialogu tisku vyber `Uložit jako PDF`.
